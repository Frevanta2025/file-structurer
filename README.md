# File Structurer

Turn messy files into clean, structured batches in seconds.

## 🔒 Privacy-first by design

- Runs entirely in your browser
- No files are uploaded to any server
- No backend, no tracking, no hidden processing
- You can verify everything in the source code

## 🧠 What it does

- Renames files consistently
- Groups them into a folder
- Generates a `_TOC.md` file
- Packages everything into a ZIP

## ⚙️ How it works

1. Open the app
2. Drop your files
3. Choose group and options
4. Download the structured archive

## 🧩 Open Source

This project is fully open source.

You can:
- inspect the code
- run it locally
- modify it

## 🚀 Why this exists

Because file organization is:
- slow
- inconsistent
- annoying

This tool makes it fast and deterministic.

## 📁 Example output
vacation_paris/
_TOC.md

## 🛠 Tech

- Pure HTML / CSS / JavaScript
- JSZip (client-side ZIP generation)
- No backend
